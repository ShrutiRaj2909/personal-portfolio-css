# Personal Portfolio Website

## Project Overview

This project is a responsive Personal Portfolio Website developed using HTML5 and CSS3 as part of the Week 2 Web Development Internship. The website presents personal information, technical skills, projects, and a contact form through a clean and user-friendly interface.

---

## Features

- Responsive web design
- Navigation menu
- About Me section
- Technical Skills section
- Projects section
- Contact form
- Hover effects on buttons and navigation links
- Google Fonts integration
- Flexbox layout
- Mobile-friendly interface

---

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Poppins)
- Visual Studio Code
- Git & GitHub

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open the `index.html` file.
4. Right-click the file and select **Open with Live Server**.
5. The website will open in your default browser.

---

## Responsive Design

The website is designed using CSS Flexbox and Media Queries to ensure proper display on desktop, tablet, and mobile devices.
